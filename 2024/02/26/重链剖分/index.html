重链剖分

洛谷P3384：重链剖分加线段树

  • 重儿子：父亲节点的所有儿子中子树结点数目最多（size最大）的结点；

  • 轻儿子：父亲节点中除了重儿子以外的儿子；

  • 重边：父亲结点和重儿子连成的边；

  • 轻边：父亲节点和轻儿子连成的边；

  • 重链：由多条重边连接而成的路径；

  • 轻链：由多条轻边连接而成的路径；

  • $siz$ : 树的大小

  • $dep$ : 点的深度

  • $top$ : 点重链的链头，如果是轻链则是本身

  • $praent$ : 点的父亲节点

  • $in$ : 点的进入时间(第一次出现的dfs序)

  • $out$ : 点的出去时间(最后一次出现的dfs序)

  • $seq$ : $in$对应的点

  • $adj$ : 为邻接表，第一个是重儿子

如果为边权，需要边权转点权，即将边权给予远离根的节点，计算边权之和时去掉转后的点权去掉$lca$即可
